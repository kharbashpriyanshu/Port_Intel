# PortIntel

Intelligent Network Reconnaissance and Port Analysis Tool

## v3.0 Features
- Single Host Scan
- Port Range Scan
- Open Port Detection
- Multi-threading for high-speed scanning
- **Service Detection (e.g., HTTP, SSH, FTP)**

## Usage
```bash
python portintel.py --target 127.0.0.1 --start 1 --end 65535 --threads 100
```
